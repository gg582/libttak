Calculates a 32-\/bit checksum for the memory header. 


\begin{DoxyItemize}
\item This implementation breaks the serial XOR chain into two independent accumulators (sum1, sum2) to encourage Instruction Level Parallelism (ILP). By breaking the data dependency, even a simple compiler like TCC can produce assembly that utilizes multiple execution units simultaneously, making the optimization compiler-\/agnostic.
